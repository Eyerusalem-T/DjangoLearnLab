# Advanced Django REST Framework API Project

A complete Django REST Framework project with advanced features including filtering, searching, ordering, permissions, and comprehensive testing.

## Features

- ✅ Django 4.2.7 and Django REST Framework 3.14.0
- ✅ Two models: Author and Book with proper relationships
- ✅ Custom serializers with validation
- ✅ Advanced filtering using django-filter
- ✅ Search functionality across multiple fields
- ✅ Ordering capabilities
- ✅ Permission classes for role-based access control
- ✅ Comprehensive unit tests
- ✅ Separate test database configuration
- ✅ RESTful API endpoints
